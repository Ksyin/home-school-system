HomeSchool

Firebase + Netlify production starter

Parent, tutor, and student portals in one system.

This package uses Firebase Authentication for login, Cloud Firestore for live data, and Firebase Storage for files such as assignments, portfolios, and learning materials.

Tutor uploads learning materials, lesson plans, assignments, assessments, and reports.
Student logs in, sees work, submits files, and tracks progress.
Parent monitors children, attendance, assessments, portfolios, and report cards.

Get started

First create your Firebase project, paste the config into assets/js/firebase-config.js, then deploy to Netlify.
